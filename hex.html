Hex → Decimal

Converta números hexadecimais para decimal decompondo cada dígito pelo seu peso.

16

Como funciona

O sistema hexadecimal (base 16) usa os dígitos 0–9 e as letras A–F, onde A=10, B=11, C=12, D=13, E=14, F=15.

Cada posição tem um peso que é uma potência de 16:

Posição Peso Valor
1º (direita)16⁰1
2º16¹16
3º16²256
4º16³4096

Para converter, multiplique cada dígito pelo seu peso e some os resultados.
Ex: 1A₁₆ = 1×16 + 10×1 = 26

No campo resultado você pode digitar expressões como 10*16.
